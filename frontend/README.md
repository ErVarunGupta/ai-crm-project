# AI-First CRM (HCP Interaction)

## Features
- AI-based form filling
- LangGraph tools (log, edit, summary, follow-up, history)
- Redux state management
- FastAPI backend

## Tech Stack
React, Redux, FastAPI, LangGraph, Groq LLM

## How to Run
# backend
uvicorn main:app --reload

# frontend
npm install
npm start
